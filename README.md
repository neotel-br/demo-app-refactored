# Demo App
Demo app refactored

## Requirements
- python3
- virtualenv(optional)

## How to Run Project

### Create virtual environment(optional)
    virtualenv venv

### Install requirements.txt
    pip install -r requirements.txt

### Migrate
    python3 manage.py makemigrations rh
    python3 manage.py migrate

### Run project
    python3 manage.py runserver

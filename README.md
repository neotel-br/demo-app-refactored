# Demo App
Demo app refactored

## Requirements
- python3

## How to Run Project

### Install requirements.txt
    pip install -r requirements.txt

### Set .env file
    touch .env
    
## Edit file
    SECRET_KEY=*GENERATE A SECRET KEY*
    DEBUG=TRUE
    CTS_IP=
    CTS_PORT=

Secret key generator: https://djecrety.ir/
    
### Run container
    docker-compose up

Test on 0.0.0.0:8000
